Jerksdle

Jerksdle is a clone of a pretty well known word guessing game. The name is an ode to my softball team, named the Jerks of course. The game is built in html, css and vanilla Javascript as a part of a course on Frontend Masters where I was revisting some of the basics of web development in order to just review the fundamentals.
